Write a Java program which creates a graphical interface, of which multiple instances can be used to simulate a group SMS/text conversation.
It should provide the ability to enter text messages and transmit them to other chatting users, and to display the chat conversation.

For maximum credit:

support an arbitrary (minimum of 2) number of chatting users
identify (display) the source of each message in the conversation.  
make use of good design principles and the appropriate pattern(s)

Have all three classes open in IntelliJ
1. Run the server class to start the server.
2. Run the client class to create an instance of a client(Run as many times you want in order for 1 or more clients).

Need to create a GUI
Need to create a Driver class
